Input de Informação

A função input()

A função input() pausa o programa e aguarda o usuário digitar algo. O valor digitado é sempre retornado como string.


  • Aceita um argumento de prompt (mensagem)
  • Lê até o usuário pressionar Enter
  • Retorna tipo str sempre
  • Pode ser combinada com conversão de tipo
# Sintaxe básica
variavel = input("mensagem: ")

# Exemplos
nome = input("Seu nome: ")
idade = input("Sua idade: ")

print("Olá, " + nome)
print(f"Você tem {idade} anos")

# ⚠ idade é str, não int!
type(idade)  # <class 'str'>

⚠ Atenção: input() sempre retorna str. Para fazer cálculos, converta o valor!

Exemplos Práticos

Do input ao resultado

💰 Calculadora de troco

preco   = float(input("Preço: R$ "))
pago    = float(input("Pagou:  R$ "))
troco   = pago - preco
print(f"Troco: R$ {troco:.2f}")
▶ saída (ex: preço=7.50, pagou=10.00)
Preço: R$ 7.50 Pagou: R$ 10.00 Troco: R$ 2.50

📐 Área do círculo

import math
r = float(input("Raio: "))
area = math.pi * r ** 2
print(f"Área = {area:.4f}")
▶ saída (ex: raio=5)
Raio: 5 Área = 78.5398

🎂 Verificador de maioridade

idade = int(input("Sua idade: "))
maior = idade >= 18
print("Maior de idade:", maior)
▶ saída (ex: idade=20)
Sua idade: 20 Maior de idade: True

🔢 Verificar número par

n = int(input("Digite um número: "))
par = n % 2 == 0
print("É par?", par)
▶ saída (ex: n=8 e n=7)
Digite um número: 8 É par? True
Digite um número: 7 É par? False

📊 Média de notas

n1 = float(input("Nota 1: "))
n2 = float(input("Nota 2: "))
media = (n1 + n2) / 2
print(f"Média: {media}")
▶ saída (ex: 8.0 e 6.0)
Nota 1: 8.0 Nota 2: 6.0 Média: 7.0
Exemplo Completo

Unindo tudo: IMC

# Calculadora de IMC — exemplo completo
# input, conversão, operadores aritméticos e lógicos

nome   = input("Nome: ")
peso   = float(input("Peso (kg): "))
altura = float(input("Altura (m): "))

imc    = peso / (altura ** 2)

baixo_peso   = imc < 18.5
normal       = imc >= 18.5 and imc < 25.0
sobrepeso    = imc >= 25.0 and imc < 30.0
obesidade    = imc >= 30.0

print(f"\n--- Resultado para {nome} ---")
print(f"IMC: {imc:.2f}")
print(f"Baixo peso?  {baixo_peso}")
print(f"Normal?      {normal}")
print(f"Sobrepeso?   {sobrepeso}")
print(f"Obesidade?   {obesidade}")
▶ saída (ex: Ana, 70kg, 1.70m)
Nome: Ana Peso (kg): 70 Altura (m): 1.70 --- Resultado para Ana --- IMC: 24.22 Baixo peso? False Normal? True Sobrepeso? False Obesidade? False

✅ Conceitos: input(), float(), **, /, >=, and, f-strings
