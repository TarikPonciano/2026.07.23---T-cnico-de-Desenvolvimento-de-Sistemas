O que são Estruturas de Decisão?

Na vida real, tomamos decisões o tempo todo com base em condições:

  • Se estiver chovendo, levo guarda-chuva.
  • Se a nota for ≥ 7, aprovado; senão, reprovado.
  • Se o saldo for positivo, exibo o valor; caso contrário, exibo alerta.

Em Python, usamos as palavras-chave if, elif e else para traduzir essa lógica para o código. Elas permitem que o programa execute blocos diferentes dependendo do resultado de uma expressão booleana (True ou False).

Sem estruturas de decisão, um programa sempre faria a mesma coisa — seria linear e sem inteligência.

O if Simples

O if avalia uma expressão booleana. Se o resultado for True, o bloco indentado é executado. Se for False, o bloco é simplesmente ignorado.

Sintaxe

if condição:
    # bloco executado se True
    instrução_1
    instrução_2

# código fora do if (sempre executa)

⚠ Indentação é obrigatória

Python usa 4 espaços (ou 1 Tab) para definir o escopo. Errar a indentação causa IndentationError.

Exemplo Prático

temperatura = 38.5

if temperatura > 37.5:
    print("Febre detectada!")
    print("Procure um médico.")

print("Verificação concluída.")

Saída

Febre detectada!
Procure um médico.
Verificação concluída.

Se temperatura = 36.0, as duas primeiras linhas não seriam impressas.

O if / else

O else define o bloco que será executado quando a condição do if for False. Garante que sempre um dos dois caminhos seja percorrido.

if condição:
    # executa se True
    bloco_A
else:
    # executa se False
    bloco_B

Dica Didática

Pense no else como a tradução literal de "caso contrário". O else não tem condição — ele é o "balde" que captura tudo o que o if não capturou.

Exemplo — Aprovação Escolar

nota = float(input("Digite a nota: "))

if nota >= 7.0:
    print("✅ Aprovado!")
else:
    print("❌ Reprovado.")

Exemplo — Par ou Ímpar

numero = int(input("Digite um número: "))

if numero % 2 == 0:
    print(numero, "é PAR")
else:
    print(numero, "é ÍMPAR")

O operador % retorna o resto da divisão. Se o resto por 2 for 0, o número é par.

O elif — Múltiplas Condições

Quando há mais de dois caminhos possíveis, usamos elif (abreviação de "else if"). Python avalia as condições de cima para baixo e executa apenas o primeiro bloco verdadeiro.

nota = float(input("Nota: "))

if nota >= 9.0:
    print("Conceito A — Excelente")
elif nota >= 7.0:
    print("Conceito B — Bom")
elif nota >= 5.0:
    print("Conceito C — Regular")
else:
    print("Conceito D — Insuficiente")

Como Python avalia

  • nota = 9.5 → bloco if → "Conceito A"
  • nota = 8.0 → if falso → elif → "Conceito B"
  • nota = 5.5 → dois falsos → 2º elif → "Conceito C"
  • nota = 3.0 → todos falsos → else → "Conceito D"

Regra de Ouro

Apenas um bloco é executado. Assim que uma condição verdadeira é encontrada, Python pula todos os outros elif e else.

Quantidade

Você pode ter quantos elif quiser, mas apenas um if e um else por estrutura.

Operadores de Comparação e Lógicos

Comparação — retornam True/False

==   # igual a
!=   # diferente de
>    # maior que
<    # menor que
>=   # maior ou igual
<=   # menor ou igual

Lógicos — combinam condições

and  # AMBAS devem ser True
or   # PELO MENOS UMA True
not  # INVERTE o resultado

Exemplos Combinados

idade = 20
tem_habilitacao = True

# and: as duas precisam ser True
if idade >= 18 and tem_habilitacao:
    print("Pode dirigir!")

hora = 22

# or: basta uma ser True
if hora < 6 or hora > 22:
    print("Silêncio — hora de dormir.")

logado = False

# not: inverte
if not logado:
    print("Faça login primeiro!")

and

T and T = True
T and F = False
F and F = False

or

T or F = True
F or T = True
F or F = False

not

not True = False
not False = True

Estruturas Aninhadas (nested)

É possível colocar um if dentro de outro if. Isso permite verificar condições em cascata — situações em que uma segunda verificação só faz sentido se a primeira já for verdadeira.

idade = 17
acompanhado = True

if idade >= 18:
    print("Acesso liberado.")
else:
    # if aninhado dentro do else
    if acompanhado:
        print("Menor — acesso com responsável.")
    else:
        print("Acesso negado.")

⚠ Cuidado com o aninhamento excessivo

Muitos níveis de indentação tornam o código difícil de ler. Quando isso acontece, considere usar operadores lógicos (and, or) para simplificar, ou extrair lógica para funções.

Equivalente com and

# Versão mais limpa
if idade < 18 and acompanhado:
    print("Menor — acesso com responsável.")
elif idade >= 18:
    print("Acesso liberado.")
else:
    print("Acesso negado.")

O match / case

Introduzido no Python 3.10, o match/case é a resposta pythônica ao switch/case de outras linguagens. Ideal quando você compara um valor contra múltiplas opções fixas.

dia = int(input("Dia da semana (1-7): "))

match dia:
    case 1:
        print("Segunda-feira")
    case 2:
        print("Terça-feira")
    case 6 | 7:  # múltiplos valores
        print("Final de semana! 🎉")
    case _:      # coringa (padrão)
        print("Dia de semana comum")

Destaques do match/case

  • O _ (underscore) funciona como o else — captura qualquer valor não correspondido
  • O | permite listar múltiplos valores no mesmo case
  • Muito mais legível que muitos elif consecutivos
  • Suporta pattern matching avançado (tuplas, classes)

Quando usar?

Prefira match/case quando comparar o mesmo valor contra várias opções discretas. Use if/elif para intervalos numéricos ou condições compostas.

Calculadora de IMC

Vamos unir tudo que aprendemos em um programa real que calcula o Índice de Massa Corporal e classifica o resultado.

# ── Calculadora de IMC ──────────────────
peso = float(input("Peso (kg): "))
altura = float(input("Altura (m): "))

if altura <= 0 or peso <= 0:
    print("❌ Valores inválidos!")
else:
    imc = peso / (altura ** 2)
    print(f"IMC: {imc:.2f}")

    if imc < 18.5:
        print("Abaixo do peso")
    elif imc < 25.0:
        print("✅ Peso normal")
    elif imc < 30.0:
        print("Sobrepeso")
    elif imc < 35.0:
        print("Obesidade grau I")
    else:
        print("Obesidade grau II ou III")

O que este programa usa?

  • if simples para validar entradas
  • else para o caminho principal
  • if/elif/else aninhado para classificar
  • Operador lógico or para checar dois erros de uma vez
  • f-string com formatação :.2f para 2 casas decimais

Tabela OMS — IMC

< 18.5 → Abaixo do peso
18.5 – 24.9 → Normal
25.0 – 29.9 → Sobrepeso
30.0 – 34.9 → Obesidade I
≥ 35.0 → Obesidade II/III
