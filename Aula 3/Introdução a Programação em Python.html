Primeiros passos

Comando print()

O print() é usado para exibir informações na tela. É a primeira função que todo programador aprende!

# Exibindo uma mensagem simples
print("Olá, Mundo!")

# Exibindo números
print(42)      # inteiro
print(3.14)    # decimal

# Exibindo várias coisas juntas
print("Resultado:", 10 + 5)   # → Resultado: 15

💡 Tudo que está dentro dos parênteses será mostrado no terminal.

Conceito

Variáveis

Uma variável é um espaço na memória com um nome para armazenar valores. Pense como uma caixa com etiqueta.

# Criando variáveis
nome     = "Ana"
idade    = 17
nota     = 9.5
aprovado = True

print(nome)    # Ana
print(idade)   # 17

str – texto   ex: "Ana"

int – inteiro   ex: 17

float – decimal   ex: 9.5

bool – lógico   ex: True / False

Interação com usuário

Comando input()

O input() permite que o programa receba dados digitados pelo usuário no terminal.

# Lendo o nome do usuário
nome = input("Digite seu nome: ")
print("Olá,", nome)

# Lendo um número (converter com int)
idade = int(input("Qual sua idade? "))
print("Você tem", idade, "anos.")

⚠️ Atenção: o input() sempre retorna texto (str). Para usar como número, use int() ou float().

Montando textos

Exibindo variáveis e montando textos

Três formas de montar textos com variáveis em Python:

nome  = "Carlos"
idade = 18

# 1. Vírgula no print
print("Aluno:", nome, "- Idade:", idade)

# 2. Concatenação com +
print("Aluno: " + nome + " - Idade: " + str(idade))

# 3. f-string  ← mais moderno e recomendado ✨
print(f"Aluno: {nome} - Idade: {idade}")

💡 A f-string é a forma mais elegante: coloque f antes das aspas e use {variavel} direto no texto.

f-string em detalhe

Formatações com f-string

preco  = 9.5
nome   = "ana"
nota   = 8.333333
qtd    = 42

# Decimais: .2f → 2 casas
print(f"Preço: R$ {preco:.2f}")
# → Preço: R$ 9.50

# Arredondar nota
print(f"Nota: {nota:.1f}")
# → Nota: 8.3

# Maiúsculas / minúsculas
print(f"{nome.upper()}")   # → ANA
print(f"{nome.title()}")   # → Ana

# Largura mínima (alinhamento)
print(f"{qtd:05d}")   # → 00042

📝 Referência rápida

{valor:.2f} 2 casas decimais
{valor:.0f} sem decimal (arredonda)
{texto.upper()} TUDO MAIÚSCULO
{texto.lower()} tudo minúsculo
{texto.title()} Primeira Maiúscula
{num:05d} preenche com zeros
{num:,} separador de milhar

💡 Você pode usar expressões direto no {}: f"{2 + 2}" → 4

print() avançado

Impressão multi-linha

Às vezes precisamos exibir textos longos ou formatados em várias linhas. Python oferece três formas:

# 1. \n dentro da string — pula linha
print("Linha 1\nLinha 2\nLinha 3")

# 2. Vários print() seguidos
print("Nome:   Ana")
print("Turma:  TDS")
print("Cidade: Fortaleza")

# 3. Triple-quote — exibe exatamente como escrito
print("""
=== FICHA DO ALUNO ===
Nome  : Ana
Turma : TDS
Nota  : 9.5
======================
""")

🔄 Caracteres especiais em strings

\n nova linha
\t tabulação (tab)
\\ barra invertida
\" aspas dentro da string

💡 Triple-quote + f-string

print(f"""
Olá, {nome}!
Nota: {nota:.1f}
""")

Atividades da Aula

Praticar print(), variáveis, input(), f-strings e formatação de saída. Foco total em coletar e exibir dados — sem estruturas de decisão.

Atividade 1 de 8
🟥 Fácil

Apresentação pessoal — ativ01.py

📝 O que fazer

Em seguida, use print() com f-string para exibir cada informação em uma linha separada, com uma etiqueta antes do valor.

🔧 Conceitos: declaração de variáveis str/int, múltiplos print(), f-string básica com {variavel}

Conversor de temperatura — ativ04.py
